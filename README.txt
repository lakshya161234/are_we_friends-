Friendship Sunset Site
======================

How to run:
1. Extract the zip.
2. Open index.html in any modern browser.

No installation is required.

Files:
- index.html
- styles.css
- script.js

What it does:
- Asks: "Has our friendship ended?"
- The Yes button runs away on desktop and phone.
- Clicking No launches a celebration with confetti and fireworks.
- Cute sunset-themed design in reddish-yellow tones.
